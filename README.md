# DAP391m_project
These Notebooks are writed to run on Google Colab Notebook. 
To run code, you should upload them to Colab and add Fake Job Posting Dataset to /content/drive/MyDrive/Colab Notebooks with filename fake_job_postings.csv
Dataset Link: https://www.kaggle.com/datasets/shivamb/real-or-fake-fake-jobposting-prediction
You must run DAP391m_process (to convert text features to vectors) before running DAP391m_train
You can also run these on local computer, but you need to edit something.
